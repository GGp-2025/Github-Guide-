GitHub Guide: Account, Project, Hosting

1. How to Create a GitHub Account

Step 1: Go to GitHub and click "Sign up".

Step 2: Fill in your details and complete verification.

2. How to Post a Project on GitHub

Step 1: Click on the "+" icon and select "New Repository".

Step 2: Upload files and commit changes.

3. How to Host Your Project on GitHub Pages

Step 1: Create a repository with the name "your-username.github.io".

Step 2: Upload your HTML, CSS, and JS files, then enable GitHub Pages in Settings.
